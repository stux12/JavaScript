node.js : javascript를 이용한 프레임워크 (서버쪽을 하기위해 백엔드)
네트워크상 서비스를 할수있는 플랫폼

동기식(Blocking) : 순서대로 작업을하는것 1 > 1-1 > 2, 한작업이 끝날때 까지 기다림
비동기식(Non-Blocking) : 1 > 2 노드는 비동기식의 대표적인 예, 한작업을 시작해놓고 다른작업을 함
최근 모든 서비스는 비동기식으로 하는 추세

개발환경설정
https://nodejs.org/ko/로 들어가서 LTS버전으로 쓰는게 좋음
또한 이클립스를 통해서 함 vscode는 프론트전용
이클립스 마켓플레이스 enide검색 그중 node.js, javascript 초록색 공모양

파일 new nodejs-project
실행 해당 js파일 마오 런 -> 노드어플리케이션
실행시키고 홈페이지에 http://127.0.0.1:1337로 들어가면 서버 실행중인거 보임

웹서비스 구현 : 서버IP찾아가고 포트번호로 세부적으로 찾아감
